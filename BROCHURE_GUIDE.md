# 📘 회사 소개서 카탈로그 가이드

## 개요
09all의 모든 정보를 담은 전문적인 회사 소개서(브로셔) 시스템이 구현되었습니다.
웹에서 확인하고 A4 용지에 최적화된 형태로 인쇄할 수 있습니다.

## 주요 특징

### 1. 완전한 회사 소개서
- **페이지 수**: 11페이지
- **내용 구성**: 회사의 모든 정보를 체계적으로 정리
- **전문적 디자인**: 비즈니스 미팅이나 제안서에 바로 사용 가능

### 2. 구성 페이지

#### Page 1: 표지
- 회사명 및 로고
- 슬로건: "연구장비 매입 전문 기업"
- 핵심 통계 (20+ 년 업력, 500+ 거래처, 5,000+ 장비 공급, 99% 만족도)

#### Page 2: 목차
- 전체 페이지 구성 안내
- 페이지 번호 표시

#### Page 3: 회사 개요
- 회사 기본 정보
- 설립일, 대표자, 직원 수, 연매출 등
- 비전 및 미션

#### Page 4: 회사 연혁
- 2004년부터 현재까지의 주요 연혁
- 타임라인 형식으로 시각화
- 주요 성과 및 마일스톤

#### Page 5: 핵심 가치
- 전문성 (Expertise)
- 신뢰성 (Reliability)
- 혁신 (Innovation)
- 각 가치별 세부 설명

#### Page 6: 서비스 소개
- 매입 가능 장비 카테고리
- 주요 취급 브랜드
- 서비스 영역 (대학, 연구소, 기업 등)

#### Page 7: 매입 프로세스
- 5단계 프로세스 상세 설명
- 각 단계별 소요 시간
- 전체 3-5일 이내 완료

#### Page 8: 매입 가이드
- 매입 전 준비사항
- 평가 기준
- 가격 책정 방식
- 매입 성공 팁

#### Page 9: 배송 및 수거
- 전국 무료 수거 서비스
- 지역별 소요 시간 및 비용
- 안전한 포장 및 운반 방법

#### Page 10: 주요 실적
- 주요 거래처 목록
- 최근 매입 실적 사례
- 신뢰할 수 있는 실적 증명

#### Page 11: 연락처
- 전화, 팩스, 이메일, 카카오톡
- 주소 및 오시는 길
- 운영 시간

## 사용 방법

### 웹에서 확인하기
1. 상단 메뉴 → "회사소개" → "회사 소개서" 클릭
2. 또는 직접 `/brochure` 페이지 접속
3. 스크롤하여 전체 내용 확인

### 인쇄하기
1. 소개서 페이지 우측 상단의 "카탈로그 인쇄" 버튼 클릭
2. 또는 Ctrl+P (Windows) / Cmd+P (Mac)
3. 인쇄 설정:
   - **용지**: A4
   - **방향**: 세로
   - **여백**: 없음 또는 최소
   - **배경 그래픽**: 켜기 (색상 출력 필수)
   - **페이지 범위**: 전체 또는 필요한 페이지만 선택

### PDF로 저장하기
1. 인쇄 대화상자에서 "프린터"를 "PDF로 저장" 선택
2. 저장 위치 선택
3. 파일명 입력 (예: 09all_회사소개서_2026.pdf)
4. 저장

## 문제 해결

### 인쇄 시 색상이 안 나와요
- 브라우저 인쇄 설정에서 "배경 그래픽" 활성화
- Chrome: "기타 설정" → "배경 그래픽" 체크

### 페이지가 잘려요
- 브라우저 확대/축소를 100%로 설정
- 용지 크기를 A4로 설정
- 여백을 "없음" 또는 "최소"로 설정

### PDF 저장이 안 돼요
- Chrome, Edge 등 최신 브라우저 사용
- "프린터"에서 "PDF로 저장" 옵션 선택
- 파일 저장 권한 확인

### 모바일에서 보기 불편해요
- 소개서는 데스크톱 및 인쇄용으로 최적화됨
- 모바일에서는 각 페이지별로 스크롤하여 확인
- PDF로 저장하여 모바일에서 확인 권장
