# 📚 장비 카탈로그 가이드

## 개요
웹 기반 장비 카탈로그 시스템이 구현되었습니다. 사용자는 전체 장비 목록을 한눈에 확인하고 인쇄할 수 있습니다.

## 주요 기능

### 1. 웹 카탈로그 뷰
- **URL**: `/catalog`
- 모든 장비를 카탈로그 형태로 깔끔하게 정리
- 그리드 레이아웃으로 한눈에 보기 쉬운 구성

### 2. 고급 필터링
- **카테고리 필터**: 분석장비, 계측장비, 실험장비 등
- **브랜드 필터**: Agilent, Thermo Fisher 등 브랜드별 분류
- **정렬 옵션**: 카테고리순, 제품명순, 브랜드순

### 3. 인쇄 최적화
- **원클릭 인쇄**: 인쇄 버튼 클릭 하나로 전문적인 카탈로그 출력
- **A4 페이지 최적화**: 인쇄 시 자동으로 A4 용지에 맞춰 레이아웃 조정
- **2열 레이아웃**: 인쇄 시 2열 그리드로 자동 변환
- **페이지 나누기 방지**: 장비 정보가 페이지 중간에서 잘리지 않도록 최적화
- **인쇄 전용 헤더/푸터**: 
  - 회사 정보
  - 연락처
  - 발행일자
  - 웹사이트 URL

### 4. 카탈로그 카드 정보
각 장비 카드에는 다음 정보가 포함됩니다:

- 순번 및 카테고리
- 브랜드 및 제품명
- 모델 및 제조년도
- 상태 (신품/리퍼비시/데모용)
- 보증 기간
- 제품 설명
- 주요 사양 (최대 3개)
- 주요 특징 (최대 3개)
- 가격 정보
- 연락처 및 상세 페이지 링크

## 사용 방법

### 웹에서 사용
1. 상단 메뉴 → "매입 가능 장비" → "장비 카탈로그" 클릭
2. 또는 직접 `/catalog` 페이지 접속
3. 필터와 정렬 옵션으로 원하는 장비 찾기
4. 필요시 "초기화" 버튼으로 필터 리셋

### 인쇄하기
1. 카탈로그 페이지에서 원하는 필터 적용
2. "인쇄" 버튼 클릭 (또는 Ctrl+P / Cmd+P)
3. 인쇄 설정에서 다음 권장:
   - 용지: A4
   - 방향: 세로
   - 여백: 기본 또는 최소
   - 배경 그래픽: 켜기 (색상 출력 시)

## 스타일링

### 화면용 레이아웃
- 모바일: 1열 그리드
- 태블릿: 2열 그리드
- 데스크톱: 3열 그리드

### 인쇄용 레이아웃
- A4 용지에 2열 그리드
- 15mm 여백
- 페이지 나누기 최적화
- 불필요한 UI 요소 숨김 (헤더, 필터, 버튼 등)

## 향후 개선 사항 (선택사항)

### 단계 2: PDF 생성
- 서버 사이드 PDF 생성
- 다운로드 가능한 PDF 파일
- 이메일 전송 기능

### 단계 3: 고급 기능
- QR 코드 자동 생성 (각 장비 상세 페이지 링크)
- 커스텀 카탈로그 (원하는 장비만 선택하여 카탈로그 생성)
- 다국어 카탈로그 (영문 버전)
- 로고 및 워터마크 추가

## 기술 스택
- **React Server Components**: 서버에서 데이터 fetch
- **Client Components**: 인터랙티브 필터링 및 정렬
- **CSS Print Media Queries**: 인쇄 최적화
- **Responsive Design**: 모든 기기에서 완벽한 표시

## 문제 해결

### 인쇄 시 색상이 안 나와요
- 브라우저 인쇄 설정에서 "배경 그래픽" 또는 "Background graphics" 옵션 활성화

### 인쇄 시 레이아웃이 깨져요
- 브라우저 확대/축소 비율을 100%로 설정
- 용지 크기를 A4로 설정
- 여백을 기본값으로 설정

### 필터가 작동하지 않아요
- 페이지를 새로고침해보세요
- 브라우저 콘솔에서 에러 메시지 확인
